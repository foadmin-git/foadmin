### ✨ 项目简介

Foadmin 是一款现代化的企业级后台管理系统，基于 **Vue 3** 和 **Python FastAPI** 构建。我们致力于为开发者提供高效、灵活、易用的管理后台解决方案。

### 🛠️ 技术栈

| 层级 | 技术 | 说明 |
|:---:|:---:|:---|
| 前端框架 | Vue 3 + Vite 7 | 渐进式 JavaScript 框架 |
| UI 组件 | Element Plus | 基于 Vue 3 的组件库 |
| CSS 方案 | Tailwind CSS | 原子化 CSS 框架 |
| 图表 | ECharts | 数据可视化图表库 |
| 富文本 | WangEditor | 轻量级 Web 富文本编辑器 |
| 后端框架 | FastAPI | 高性能异步 Python Web 框架 |
| ORM | SQLAlchemy 2.0 | Python SQL 工具包与 ORM 框架 |
| 数据库 | MySQL | 关系型数据库 |
| 认证 | JWT (python-jose) | 无状态令牌认证 |
| 定时任务 | APScheduler | 高级 Python 定时任务调度器 |

### 🚀 快速开始

#### 环境要求

- Python 3.10+
- Node.js 18+
- MySQL 8.0+

#### 前端

```bash
cd admin
npm install
npm run dev
```

#### 后端

```bash
cd server
pip install -r requirements.txt
# 配置 .env 文件
python -m app.main
```
